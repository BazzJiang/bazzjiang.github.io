部署ghost

为了让你的ghost博客能够一直运行，我们需要用到一个工具叫做forever。

  • 通过cnpm install forever -g安装 forever
  • 为了让 forever 从 Ghost 安装目录运行，输入 NODE_ENV= production forever start index.js，记住一定要以产品模式启动。
  • 通过 forever stop index.js 停止 Ghost
  • 通过 forever list 或者ps -ef|grep index检查 Ghost 当前是否正在运行
